# AI Product Research Platform

A modular, multi-agent platform for automated product research, market analysis, and opportunity scoring. The system combines curated knowledge bases with specialized AI agents to discover, validate, analyze, and score product ideas across multiple domains.

## Project Structure

| Directory      | Purpose                                           |
|----------------|---------------------------------------------------|
| `docs/`        | Architecture and design documentation              |
| `knowledge/`   | Curated domain knowledge (market, economics, etc.) |
| `prompts/`     | Prompt templates for AI agents                     |
| `agents/`      | AI agent configurations and implementations        |
| `workflows/`   | Multi-agent orchestration workflows                |
| `database/`    | Database schema and migrations                     |
| `backend/`     | Python + FastAPI backend                           |
| `frontend/`    | React-based frontend                               |
| `scoring/`     | Scoring models and evaluation logic                |
| `reports/`     | Generated reports and templates                    |
| `tests/`       | Automated tests                                    |

See [`docs/AI_Product_Research_Platform_Architecture.md`](docs/AI_Product_Research_Platform_Architecture.md) for the full architecture and [`docs/Implementation_Roadmap.md`](docs/Implementation_Roadmap.md) for the sprint plan.
